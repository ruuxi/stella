# Notes

Daily task summaries go here, one file per day (`YYYY-MM-DD.md`).

Each entry is the General agent's final summary appended after task completion — what was done, what was learned, what's unresolved.

## Rules

- Append-only. Never modify a past day's entry.
- Entries are written automatically at task completion.
- Promote durable conclusions into `knowledge/` via the [dream protocol](state/DREAM.md).
- Keep entries terse — they're for the agent's future self, not documentation.
